## Quick Setup

### Connect to Claude Desktop

1. **Get MCP Server URL**: Navigate to your Mintlify dashboard and find your MCP server URL
2. **Configure Claude**: Add the MCP server to Claude Desktop
3. **Start Using**: Ask Claude questions about our API and documentation

<Steps>
  <Step title="Open Claude Desktop">
    Launch Claude Desktop application
  </Step>
  <Step title="Access Settings">
    Go to Settings → Developer → Model Context Protocol
  </Step>
  <Step title="Add Server">
    Click "Add Server" and enter:
    - **Name**: Visa Direct SDK
    - **URL**: `https://your-domain.mintlify.app/mcp`
  </Step>
  <Step title="Test Connection">
    Start a new conversation and ask: "What are the main Visa Direct payout endpoints?"
  </Step>
</Steps>

### Connect to Cursor

1. **Open Command Palette**: `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows)
2. **Search MCP**: Type "Open MCP settings"
3. **Add Server**: Configure your MCP server in the settings

```json
{
  "servers": {
    "visa-direct-sdk": {
      "type": "http",
      "url": "https://your-domain.mintlify.app/mcp"
    }
  }
}
```

### Connect to VS Code

1. **Install Extension**: Install the MCP extension for VS Code
2. **Configure Server**: Add server configuration to `.vscode/mcp.json`

```json
{
  "servers": {
    "visa-direct-sdk": {
      "type": "http",
      "url": "https://your-domain.mintlify.app/mcp"
    }
  }
}
```

### Advanced Usage

<CodeGroup>

```typescript TypeScript
// Ask Claude to generate code with MCP context
// "Generate a complete payout service class using the Visa Direct SDK"

import { SecureHttpClient, PayoutBuilder } from 'visa-direct-sdk';

export class PayoutService {
  private builder: PayoutBuilder;

  constructor(config: PayoutConfig) {
    const http = new SecureHttpClient({
      baseUrl: config.baseUrl,
      certPath: config.certPath,
      keyPath: config.keyPath,
      caPath: config.caPath
    });
    
    this.builder = PayoutBuilder.create(http);
  }

  async executeCardPayout(request: CardPayoutRequest): Promise<PayoutReceipt> {
    return await this.builder
      .forOriginator(request.originatorId)
      .withFundingInternal(request.fundingConfirmed, request.confirmationRef)
      .toCardViaAlias(request.alias, request.aliasType)
      .forAmount(request.currency, request.amountMinor)
      .withIdempotencyKey(request.idempotencyKey)
      .execute();
  }
}
```

```python Python
# Ask Claude to generate Python code with MCP context
# "Create a Python service class for Visa Direct payouts"

from visa_direct_sdk.transport.secure_http_client import SecureHttpClient
from visa_direct_sdk.dx.builder import PayoutBuilder
from typing import Dict, Any

class PayoutService:
    def __init__(self, config: Dict[str, str]):
        http = SecureHttpClient(
            base_url=config['base_url'],
            cert_path=config['cert_path'],
            key_path=config['key_path'],
            ca_path=config['ca_path']
        )
        self.builder = PayoutBuilder.create(http)

    async def execute_card_payout(self, request: Dict[str, Any]) -> Dict[str, Any]:
        return await self.builder \
            .for_originator(request['originator_id']) \
            .with_funding_internal(request['funding_confirmed'], request['confirmation_ref']) \
            .to_card_via_alias(request['alias'], request['alias_type']) \
            .for_amount(request['currency'], request['amount_minor']) \
            .with_idempotency_key(request['idempotency_key']) \
            .execute()
```
