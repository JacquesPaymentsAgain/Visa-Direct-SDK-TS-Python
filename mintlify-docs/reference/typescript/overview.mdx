## Overview

The TypeScript SDK provides a type-safe, production-ready interface to the Visa Direct API with comprehensive error handling, idempotency, and security features.

## Installation

```bash
npm install visa-direct-sdk
```

## Quick Start

```typescript
import { SecureHttpClient, PayoutBuilder } from 'visa-direct-sdk';

const http = new SecureHttpClient({
  baseUrl: 'https://sandbox.api.visa.com',
  certPath: './certs/client.pem',
  keyPath: './certs/client.key',
  caPath: './certs/visa-ca.pem'
});

const receipt = await PayoutBuilder.create(http)
  .forOriginator('fi-001')
  .withFundingInternal(true, 'LEDGER123')
  .toCardDirect('tok_pan_411111******1111')
  .forAmount('GBP', 5000)
  .withIdempotencyKey('ts-reference-example')
  .execute();
```

## Type Safety

The TypeScript SDK provides full type safety for all operations:

```typescript
import { PayoutBuilder, PayoutRequest } from 'visa-direct-sdk';

const builder = PayoutBuilder.create(http);
const request: PayoutRequest = builder
  .forOriginator('fi-001')
  .withFundingInternal(true, 'LEDGER123')
  .toCardDirect('tok_pan_411111******1111')
  .forAmount('GBP', 5000)
  .withIdempotencyKey('typed-request')
  .build();

const receipt = await builder.execute();
```

## Error Handling

Comprehensive error handling with typed exceptions:

```typescript
try {
  const receipt = await client.payouts.build().execute();
} catch (error) {
  if (error instanceof LedgerNotConfirmed) {
    // Handle ledger confirmation failure
  } else if (error instanceof AFTDeclined) {
    // Handle AFT decline
  } else if (error instanceof ReceiptReused) {
    // Handle receipt reuse
  }
}
```
